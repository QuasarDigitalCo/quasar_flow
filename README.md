# Quasar Flow

> Your AI workspace, always up to date.

Quasar Flow is a free tool manager and installer from [Quasar Digital](https://quasardigital.co) 
that helps you set up and maintain your full AI development stack. Built for the community, 
open to everyone.

---

## What it does

- Installs and updates your entire AI stack in one place
- Detects what you already have installed and skips it
- Keeps tools like Claude Code, Codex, Obsidian, Ollama, and more up to date automatically
- Sets up your Obsidian vaults with the right folder structure and AI configuration
- Saves your API keys securely in Windows Credential Manager
- Works as a permanent tool manager, not just a one-time installer

## Getting Started

### Download
Head to the [Releases](https://github.com/QuasarDigitalCo/quasar_flow/releases) page 
and download the latest installer for your platform.

| Platform | File |
|----------|------|
| Windows  | `Quasar-Flow-Setup-x.x.x.exe` |
| macOS    | `Quasar-Flow-x.x.x.dmg` |
| Linux    | `Quasar-Flow-x.x.x.AppImage` |

### Run
Launch the installer and follow the steps. Quasar Flow will detect what you already 
have installed and only install what's missing.
